安装node，git环境

    配置Hexo

  2. 安装Hexo hexo init <文件夹名称>
    cd <文件夹名称>
    npm install
    注：
    _config.yml:配置文件
    source: 博客文档（markdown文档）
    themes: 博客主题（默认问ladespace） hexo server
    访问 http://localhost:4000即可
    • 在github中添加SSH key（推送代码认证）
      • 将在～/.ssh目录下产生的密钥添加到github中
      • 测试是否认证成功
